1. Intro
basicSlideshow is an add-on for MODX Revolution (2.0+) that gives basic Slideshow functionality.  Slides are stored as standard resources with custom TVs allowing HTML and Image-based slides.  Users can select to optionally include slide markers, or left-right arrows, for navigation
